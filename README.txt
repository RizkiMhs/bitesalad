# GreenBite Salad Website

Website promosi produk GreenBite Salad berbasis HTML, CSS, dan JavaScript murni tanpa database.

## Fitur
1. Navbar transparan yang berubah saat scroll.
2. Hero section profesional untuk branding produk.
3. Penjelasan produk dan toko.
4. Katalog 4 produk.
5. Pop-up detail produk ketika kartu produk diklik.
6. Tombol beli yang mengarah ke WhatsApp.
7. Fitur kalkulator estimasi gizi tanpa database.
8. Responsive untuk desktop dan mobile.

## Cara Menjalankan
Buka file `index.html` di browser.

## Mengubah Nomor WhatsApp
Ubah nomor pada:
- file `index.html`, bagian link `https://wa.me/...`
- file `script.js`, variabel `whatsappNumber`

Format nomor memakai kode negara tanpa tanda plus. Contoh:
6281365942803
